# 📝 To-Do Web Application

## 📌 Overview

This project is a modern and responsive To-Do Web Application developed as part of the SkillCraft Technology Web Development Internship.

The application helps users efficiently organize and manage daily tasks with features such as task creation, editing, deletion, filtering, and status tracking.

## 🚀 Features

- Add New Tasks
- Edit Existing Tasks
- Delete Tasks
- Mark Tasks as Completed
- Search Tasks
- Filter Tasks
- Responsive User Interface
- Real-Time Task Management
- Local Storage Support
- Modern Dashboard Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📷 Project Preview

The application enables users to:

- Create and manage tasks
- Organize daily activities
- Track completed and pending tasks
- Search and filter tasks easily
- Store tasks locally for persistence
